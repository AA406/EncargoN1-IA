# Arquitectura de la solución

La arquitectura propuesta considera los siguientes componentes:
1. Interfaz web o chat para recibir la consulta del usuario.
2. Orquestador del agente para coordinar clasificación, recuperación y generación.
3. Clasificador de consultas.
4. Motor de recuperación RAG.
5. Fuentes internas y externas indexadas.
6. Filtro de relevancia.
7. Módulo generador de respuesta.
8. Validador de evidencia y confianza.
9. Escalamiento a analista humano.
10. Registro de logs y trazabilidad.
